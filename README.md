# Instructions

[Video Tutorial](https://youtu.be/yk0FQN2Ykws)

**Account 1 = left profile is the id you wish to swap.**

**Account 2 = right profile is the account you wish to swap the id to.**

1. Run idSwap.exe
2. Sign into Account 1 and Account 2, enter steam guard if needed.
3. Press edit profile for both browsers.
4. Press the 'update info' button.
5. Ensure that the textbox info is correct for whatever you want to swap.
6. Press the 'Left' button and immediately after, spam click the right button a few times.
7. Your ID will be swapped to the account on the right browser.
